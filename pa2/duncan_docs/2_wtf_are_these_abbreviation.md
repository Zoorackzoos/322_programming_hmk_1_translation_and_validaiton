## AST
abstract system tree
```text
In Racket, and computer science generally, 
AST stands for Abstract Syntax Tree. It is 
a tree-based, hierarchical representation
of source code that defines the structure 
and logical flow of a program, abstracting 
away superficial syntax details like parentheses,
and in Racket, it is often expressed through syntax objects.
```
